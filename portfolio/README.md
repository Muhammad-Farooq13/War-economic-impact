# Portfolio — Interactive Web Dashboard

This directory contains the self-contained HTML/CSS/JS portfolio dashboard
that accompanies the War Economic Impact Predictor project.

## Contents

| File | Purpose |
|------|---------|
| `index.html` | Main portfolio page — project overview, methodology, results |
| `style.css` | Styling and layout |
| `script.js` | Interactive elements (charts, tabs, animations) |

## Viewing

Open `index.html` directly in any modern browser — no build step required.

For a live server (optional):

```bash
python -m http.server 8080 --directory portfolio
# Opens: http://localhost:8080
```
